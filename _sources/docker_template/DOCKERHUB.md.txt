# AusCAT DockerHub
**DockerHub** is a cloud-based registry service provided by Docker that allows users to store and share Docker images. 
</br> 
</br>
**Docker images** are pre-built, executable packages that contain everything needed to run an application or service, including code, runtime, libraries, and system tools.
